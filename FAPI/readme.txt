Запуск:
    Все что нужно для запуска, это копировать проект, папку ForumAPI к себе на компьютер
    Все зависимости установлены в виртуальном окружении, .\ForumAPI\venv

    Порядок запуска:
        1. Перейти в директорию .\ForumAPI 
        2. В командной строке написать: .\venv\Scripts\activate 
        3. Перейти в директорию .\FAPI ( DjangoProject\ForumAPI\FAPI )
        4. В командной строке написать: python manage.py runserver 

    




Подробности:

    1. Модель Phors ( Название для Вопроса на сайте )
    2. Нет смысла переопределять модель User, по крайней мере так считаю я, из-за специфики сайта
        * Запись модели User является Клиентом 
        * Клиент является Форумом, который использует API Глобального форума FAPI ( ForumAPI )
        * Сам клиент из операций может просматривать все Форы ( вопросы ), ответы и другие записи ( Темы, Пользователи клиента )
            А так-же добавлять и удалять своих пользователей клиентов
                ( Для совершений операций удаления, создания и изменения записей Форов и Ответов используется первичный ключ пользователя клиента )
        * Пользователь клиента, это пользователь на форуме клиента, который может:
            - Добавлять записи, изменять записи, удалять записи ( Записи модели Phors и Answers )
